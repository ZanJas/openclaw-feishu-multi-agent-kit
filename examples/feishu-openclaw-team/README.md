# Example: Feishu OpenClaw Team

This example shows what a real project folder can look like after you apply the kit.

It demonstrates:

- a shared `PROJECT_CONTEXT.md`
- a minimal `ARCHITECTURE.md`
- one ADR
- one story ready for execution and review

This example is intentionally small. Its purpose is to show the shape of a disciplined multi-agent project, not to be a full production deployment.
